# Safecracker

Game based on "Summer Math Camp" by Grant Fikes

Link: [https://spheniscine.github.io/dx-safecracker/] <br>
Or here (ad-supported) [https://www.interactive.onlinemathlearning.com/fun_safecracker.php]

## Objective

A secret code is set from the parameters you chose. (For a harder game, try more digits and/or a different range of digits; the '1-4' option may be harder than you think!). Try to break this secret code in as few guesses as possible.

## Play

Each turn, the dial is spun to a random candidate code. If none of the digits in the candidate match the secret code, you automatically get a chance to guess the code.

Otherwise, you have to guess what the "value" of the candidate is. The "value" is the sum of all the digits that match the secret code both in number and position. (For example, if the secret code is 123456, and the candidate is 123561, the value would be 6: the sum of the matching 1, 2, and 3.) If you are incorrect, the correct value is revealed and the dial is spun again; if you are correct, you get a chance to guess the secret code.

When guessing the code, simply type what you think the secret code is into the given textbox; if you are correct, you win the game; otherwise, the value of your guess is revealed, and the dial is spun again.

You may view the history of past spins and guesses by using the "Show History" button.

When the game ends; the number of spins it took you to win is shown; the lower this is, the better.
